PhotoEasy — Browser Photoshop Clone (Minimal Demo)

Quick start:

1. Open a terminal in this folder (`src/photoeasy`).
2. Serve files (example):

```powershell
# Simple HTTP server with Python
python -m http.server 8000
```

3. Open http://localhost:8000 in your browser.

Features in this demo:
- Multiple layers (add/delete, visibility)
- Brush and eraser (size + color)
- Basic layer move (drag with Move tool)
- Import image into a new layer
- Export flattened PNG
